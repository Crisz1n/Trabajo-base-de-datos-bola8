# Proyecto: Sistema de Gestión de Envios nacional
Descripcion: 

Desarrollo de un ecosistema de base de datos relacional diseñado para centralizar y optimizar el flujo de envíos dentro del territorio chileno. El sistema permite la gestión de remitentes y destinatarios mediante la validación de RUT, la normalización geográfica por regiones y comunas, y el seguimiento en tiempo real del estado de los paquetes, garantizando la integridad de los datos y la eficiencia en la consulta de pedidos
