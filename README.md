**Trabajo de Diseño y Modelado de Base de Datos**
# Grupo bola10
#  Sistema de Gestión de Venta de Pasajes Interurbanos

Desarrollo de un ecosistema de base de datos relacional diseñado para centralizar y optimizar la venta y reserva de boletos de bus dentro del territorio chileno. 

El sistema permite la gestión de pasajeros y cuentas de usuario mediante la validación de RUT, la normalización geográfica de los terminales por regiones y ciudades, y el control preciso de la disponibilidad de flotas (buses de 1 y 2 pisos) y viajes programados.
